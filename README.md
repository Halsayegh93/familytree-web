# FamilyTree Web — عائلة المحمدعلي

النسخة الإلكترونية من تطبيق شجرة العائلة. مرتبطة بنفس Supabase backend الخاص بتطبيق iOS.

## التشغيل

```bash
npm run dev
```

افتح http://localhost:3000

## الـ Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Supabase — نفس قاعدة البيانات والمصادقة الخاصة بتطبيق iOS
